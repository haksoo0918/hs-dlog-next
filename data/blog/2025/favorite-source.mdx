잘 사용하고 있는 자료들 중에서 추천할 만한 것들을 모아둔 글이다.  
대부분이 개발 관련된 것들이고, 그렇지 않은 것들도 추가될 예정이다.  
이 글은 지속적으로 업데이트 될 예정이다.

## font

- [D2 Coding 글꼴](https://github.com/naver/d2codingfont) - 프로그래밍 글꼴로 유명한 한글 글꼴
- [Pretendard](https://github.com/orioncactus/pretendard) - 어느 플랫폼에서든 사용할 수 있는 system-ui 대체 글꼴

## note

- [Notion](https://www.notion.com/ko) - 더 행복한 워크 스페이스
- [Obsidian](https://obsidian.md/) - 사적인 생각을 위한 자유롭고 유연한 앱

## ui

- [Lucide](https://lucide.dev/) - svg 아이콘 라이브러리
- [tailwindcss](https://tailwindcss.com/) - Utility-first CSS 프레임워크

## vs code extension

- [Prettier - Code formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) - 코드 포맷터
- [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) - Tailwind CSS 클래스 자동완성
- [Todo Tree](https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree) - TODO 주석을 트리 형태로 보여줌

## web development

- [Vite](https://ko.vite.dev/) - 웹을 위한 완벽한 빌드 툴
