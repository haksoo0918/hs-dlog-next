혹시나 하는 마음에, 당장은 모르겠지만 나중에 쓸만할까, 괜찮아 보이는 자료 모음 - 2025년 버전

개발 관련된 자료들이다. 이 자료는 계속 업데이트 될 예정이다.

## React

- [AstroPaper](https://github.com/satnaing/astro-paper) - A minimal, accessible and SEO-friendly Astro blog theme
- [share-button](https://github.com/daviddarnes/share-button) - A Web Component to share web pages using the native OS sharing options
- [reactbits](https://www.reactbits.dev/) - React Bits - Animated UI Components for React
- [react-aria](https://react-spectrum.adobe.com/react-aria/index.html) - Craft world-class accessible components with custom styles
- [React Native Paper](https://callstack.github.io/react-native-paper/) - Cross-platform Material Design for React Native
- [Neobrutalism components](https://www.neobrutalism.dev/) - A collection of neobrutalism-styled components based on shadcn/ui
- [Powerful Data-Table for React](https://data-table.openstatus.dev/) - Extensible, fast, and easy-to-use filters with tanstack table, shadcn/ui and search params via nuqs. Open Source on GitHub
- [Kibo UI](https://www.kibo-ui.com/overview) - Kibo UI is a custom registry of composable, accessible and open source components designed for use with shadcn/ui
- [Chrome Extension Boilerplate](https://github.com/Jonghakseo/chrome-extension-boilerplate-react-vite) - This boilerplate helps you create Chrome/Firefox extensions using React and Typescript

## Vue

- [swrv](https://docs-swrv.netlify.app/) - A Vue library for data fetching
- [vueframe](https://github.com/vueframe/vueframe) - Vue library for Video Frame
- [Regle](https://reglejs.dev/) - Type safe form validation for Vue.js
- [vueuse](https://vueuse.org/) - Collection of Vue Composition Utilities
- [onyx.web design system](https://onyx.schwarz/) - A design system and Vue.js component library created by Schwarz IT
